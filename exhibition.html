Exophysics Exhibition

This is the museum of explored worlds, known to display interesting emergent properties. If, on your adventures, you stumble upon uncharted worlds that you would like to share, please submit the HTML file via an email to curator໑exophysics.net.

Please include the following in your email:
  1. A name of your choice for your universe
    • No trademark infringement (google it before)
    • Must be "safe for work" and inoffensive in all lanuages
  2. The name of the discoverer, with an optional link
  3. One HTML file
    • WITH NO EXTERNAL DEPENDENCIES OF ANY KIND
    • No copyright infringement
    • Optionally with a link to GitHub, GitLab, or other code hoster
    • Your universe MUST NOT permit the existence of conscious entities capable of suffering
  4. One PNG or BMP file with a 350x250 pixel screenshot
  5. Your survey report, describing the properties of your universe
  6. The sentence: "As the sole copyright holder, I hereby release the attached data into the public domain, and allow you to publish it on https://exophysics.net"
  • Click on any of the images to teleport into the universe
    • Exophysics uses modern technologies. If it doesn't run for you, try another device or a modern browser like Firefox
  • Use mouse clicks and the keys "." and "space" to interact with particles
  • To view the source code, first click on the image, then use the browser's menu option to view the page source code. Often the shortcut for this is to press Ctrl+U
  • Feel free to change any of these universes and creating new ones by right-clicking on the image and clicking "Save link as..."
    • The resulting HTML file can be edited with a text editor
    • To start your new universe, open it in your browser, for example by double-clicking on the file
    • It's best to start with the most current version of Exophysics

Exhibits

Ogneron

  1. Code: link
  2. Discovery: 2020-10-25, by hut
  3. Properties:
    1. Forces: electric (3 charges), strong close-range force
    2. Intricate exochemistry with a multitude of configurations
    3. Large-scale molecule chains
    4. Supermassive singularities form under high pressure

Edracia

  1. Code: link
  2. Discovery: 2020-10-18, by hut
  3. Properties:
    1. Forces: gravity, electric (3 charges)
    2. 2-3 particles of different charge form stable molecules
    3. Two molecules connect and move along perpendicular to the axis of connection
    4. Forms plasma under pressure
    5. Gravity forms star-like object that evaporates, and ejects particles
    6. Particles have two velocity vectors, one per force (gravity, electric), with their own maximum velocity

    2. 1 blue + 1 red form stable molecules
    3. Forms plasma under pressure
    4. Particles have two velocity vectors, one per force (gravity, electric), with their own maximum velocity
